$\sum$ TeX / LaTeX 수식 문법 도움말

선택한 박스에 입력하는 내용이 MathJax를 통해 실시간으로 수식으로 렌더링됩니다.

📌 기본 구조 및 표기법

  • 인라인 수식: \$ \sum_{i=1}^n i \$
    결과: \(\sum_{i=1}^n i\)
  • 블록 수식: \$\$ E=mc^2 \$\$
    (새 줄에 가운데 정렬)
  • 분수: \frac{분자}{분모}
    결과: \(\frac{1}{2}\)
  • 제곱/첨자: x^2, y_i
    결과: \(x_i^2\)
  • 제곱근: \sqrt{내용}
    결과: \(\sqrt{x^2+y^2}\)
  • 행렬/벡터: \begin{pmatrix} ... \end{pmatrix}

➗ 자주 사용되는 함수/기호

  • 합계: \sum_{아래}^{위}
    결과: \(\sum_{i=0}^{\infty} i^2\)
  • 적분: \int_{아래}^{위}
    결과: \(\int_0^1 f(x)dx\)
  • 극한: \lim_{n \to \infty}
    결과: \(\lim_{x \to 0} \frac{\sin x}{x}\)
  • 이항 계수: \binom{n}{k}
    결과: \(\binom{n}{k}\)
  • 논리 기호: \forall, \exists, \land, \lor
  • 화살표: \to, \leftarrow, \leftrightarrow, \implies

도움말 닫기: F1 또는 ESC 키, 혹은 아무 곳이나 클릭

📘 스마트 트래블 맵 에디터 매뉴얼

🖱️ 기본 조작

  • 박스 선택: 클릭 (하나 선택) / Ctrl+클릭 (여러 개 추가 선택)
  • 드래그 선택: 빈 공간에서 마우스 드래그
  • 이동/크기: 박스 드래그 / 우측 하단 핸들로 크기 조절
  • 텍스트 편집: 박스 더블 클릭
  • 복제/삭제: Ctrl+D (복제), Delete (삭제)
  • 실행 취소: Ctrl+Z (취소), Ctrl+Y (다시 실행)

✨ 그룹 & 색상

  • 그룹 묶기: 여러 개 선택 후 [🔗 묶기] 버튼
  • 그룹 색상: 그룹 선택 후 상단 [🎨 색상]에서 색 선택
  • 연결선: 박스 연속 클릭 (또는 Shift+클릭)으로 연결
  • 선 꾸미기: 선 클릭(화살표), Shift+클릭(색상변경), Alt+클릭(삭제)

📐 정렬 & 배치

  • 자동 정렬: 여러 개 선택 후 상단 정렬 아이콘(⬅,║,➡,⬆...) 클릭
  • 너비 맞춤: 여러 개 선택 후 기준(빨강) 박스를 Shift+클릭 하거나 상단 [↔ 너비동일] 버튼 클릭
  • 글자 정렬: [T⇠] [T|] [T⇢] 버튼으로 텍스트 정렬 변경

📂 탭(Tab) 관리 & 저장

  • 탭 순서 변경: 탭을 드래그하여 순서 바꾸기
  • 박스 탭 이동: 박스 선택 후, 옮길 탭을 Shift+Ctrl+클릭 (위치 유지)
  • 탭 이름 변경: 탭 더블 클릭
  • 저장: Ctrl+S (프로젝트 저장) / 자동 저장 지원
  • 배경 설정: 하단 슬라이더(밝기), [🖼 배경] 이미지 삽입

💡 팁: 화면을 이동하려면 Space 키를 누른 채 드래그하거나, 상단의 [🖐 손도구] 버튼을 사용하세요.
